##main.sの作品説明

####テーマ名「星のラズビィ」


~プロローグ~

ラズベリー星のとある地方大学に通っているパイ学部2年生のラズビィは実験づけの忙しい日々をおくっていました。ある日あまりのブラックさに逃げ出したくなったラズビィは家に引きこもっていまいます。そんな中たまたま見つけた電波掲示板の天空の頂に存在する自由の宝箱の噂。得たものは一生の自由を手にすることができるという。ラズビィは希望のかけらを胸に冒険の旅にでるが。。。

####ゲームの遊び方

本ゲームは横スクロールアクションゲームです。
ディスプレイの右端から迫ってくるステージをSW1(下降)とSW3(上昇)を上手く利用してかいくぐり,ゴールである宝箱を手に入れます。

※宝箱を手に入れると晴れ晴れでゆかいな曲が流れるようになっています。

####プログラムの動作及び仕様説明
まず電源を入れるとラズビィを待ち受けにしたスタート画面が表示されます。その後SW4を押すとゲームがスタートします。プレイ画面に移るとラズビィがドットマトリックスLEDの2列目に表示されます。その後0.5秒ごとにステージが進んでいきます。ラズビィはSW1を押すと1マス下に移動(下降)、 SW3を押すと1マス上に移動(上昇)するようになっています。また途中障害物(ステージ)にあたるとゲームオーバー画面に移ります。ゴール地点までたどり着くとクリア画面に移ります。どの状況下でもSW2を押すことでスタート画面に戻ることができます。


各スイッチを押したときの動作

   SW4 ・・ スタートします

   SW2 ・・ スタート画面に戻ります

   SW1 ・・ ラズビィが下降します

   SW3 ・・ ラズビィが上昇します
