・bigdata_analysis
【松山市の気象データを対象としたデータ分析】
大学院科目のBig data analysisを受講し、グループで松山市の気象データ「校区のお天気」を用いて統計解析を行いました。私たちのグループでは「降水量は降水時間に影響を及ぼすのか、及ぼす場合には降水時間は予測できるのではないか」という疑問持ち、テーマに設定しました。
グループ内で私は、使用する変数を降水時間、降水量(0~5mm、5~10mm、10mm~)と設定することで、降水時間を従属変数、降水量を独立変数とした1要因3水準分散分析を実施に携わりました。
サンプリングサイズは中心極限定理より30個とし、ランダムサンプリングによってデータを抽出しました。以下に実際に行った統計的仮設検定の流れを記述しました。
Tを標本の総平均、Aを標本の各条件の平均とすると測定値X_ijの構造モデルは
X_ij = T+(A_j=T)+(X_ij-A_j)
で表される。このとき、データ数をnとすると群間平方和(SS_A)、群内平方和(SS_w)はそれぞれ、
SS_A=Σ_j n_j(A_j-T)^2、SS_w=Σ_jΣ_i (X_ij-A_j)^2
によってそれぞれ表される。
また、群間自由度df_Aをdf_Aをa-l、群内自由度df_wをdf_w=N-aとする場合、検定統計量Fは
F=(SS_A/(a-l))/(SS_w/(N-a))
より求まる。
その結果は、降水量の水準間における降水時間に有意な差は見られず(F(2,87)=1.72、p=0.18)、降水量は降水時間に影響を及ぼさない結果となりました。
その後は別のメンバーが降水時間に影響を及ぼす要因が存在するかを調査するために降水時間、降水量、風速、温度を変数として利用し、降水時間を従属変数、降水量、風速、温度を独立変数とした重回帰分析を実施しました。最後に結果を踏まえて考察を行いました。
